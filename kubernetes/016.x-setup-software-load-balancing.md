# Guides: Set Up Software Load Balancing

This guide is based on [Options for Software Load Balancing].

Set required values:
```shell
export STATE="MASTER"
export INTERFACE="$(ip addr | grep 192.168.56. | sed 's/.* //')"
# An ID of your choosing unique to your cluster
export ROUTER_ID=51


envsub < keepalived.conf | sudo tee /etc/keepalived/keepalived.conf
```

```shell
cat <<TXT | sudo tee -a /etc/profile.d/k8s.sh
export CLUSTER_CA_CERT=/etc/kubernetes/pki/ca.crt
export CONTROL_PLANE_PORT=6443
export CONTROL_PLANE_HOST=127.0.0.1
TXT

sudo chmod +x /etc/profile.d/k8s.sh
```
