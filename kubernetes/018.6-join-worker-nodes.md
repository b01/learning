# Join Worker Nodes

Add some worker nodes to the cluster. These will be used for regular Pod
deployments.

1. Copy the join template from the samples
   [kubeadm-dual-stack-join-node-aws.yaml] on each worker node:
   ```shell
   scp ~/samples/kubeadm-dual-stack-join-node-aws.yaml.tmpl worker-01:~/
   scp ~/samples/kubeadm-dual-stack-join-node-aws.yaml.tmpl worker-02:~/
   ```
2. Will need to fill in the placeholders in the template using `envsubst`
   like so:
   ```shell
   iface="ens5"
   export IPV4_ADDR="$(ip -4 route get "8.8.8.8" | awk '{print $7}')"
   export IPV6_ADDR="$(ip -6 route get "2607:f8b0:4009:816::200e" | awk '{print $11}')"
   echo "IPV4_ADDR=${IPV4_ADDR}"
   echo "IPV6_ADDR=${IPV6_ADDR}"
   export KUBE_TOKEN="<token>"
   export KUBE_TOKEN_SHA="<token-sha>"
   envsubst < kubeadm-dual-stack-join-node-aws.yaml.tmpl | sudo tee kubeadm-dual-stack-join-node-aws.yaml
   sudo kubeadm join --config kubeadm-dual-stack-join-node-aws.yaml
   ```

   NOTE: Get the token and its SHA from the output when you initialized the
   cluster, or make a new one with the command
   `kubeadm token create --print-join-command`.
