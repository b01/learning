# Deploy an AWS Load Balanced Service

This takes a deployment, places a Kubernetes Service in front of it, which is
backed by an AWS load balancer simply by using the type `LoadBalancer` for the
service.

NOTE: This allocates a real AWS Load Balancer, which cost money. As long as we
don't leave it running for more than 10 minutes, we should not be charged more
than a few cents on the dollar.

1. Make a deployment using Nginx:
   ```shell
   cat <<TXT | sudo tee deploy-nginx-alb-app-1.yaml
   apiVersion: apps/v1
   kind: Deployment
   metadata:
     name: nginx-alb-app-1
   spec:
     selector:
       matchLabels:
         run: nginx-alb-app-1
     replicas: 2
     template:
       metadata:
         labels:
           run: nginx-alb-app-1
       spec:
         containers:
         - name: nginx-alb-app-1
           image: nginx
           ports:
           - containerPort: 80
   TXT
   kubectl apply -f deploy-nginx-alb-app-1.yaml
   kubectl get pods -l run=nginx-alb-app-1 -o wide
   ```
2. Create a Service of type `LoadBalancer`:
   ```shell
   cat <<TXT | tee svc-nginx-alb-1.yaml
   apiVersion: v1
   kind: Service
   metadata:
     name: nginx-alb-app-1-svc
   spec:
    selector:
      run: nginx-alb-app-1
    type: LoadBalancer
    ports:
    - port: 80
   TXT
   kubectl apply -f svc-nginx-alb-1.yaml
   ```
3. Get the Service's IP address:
   ```shell
   kubectl describe services nginx-alb-app-1-svc
   kubectl get pods -l run=nginx-alb-app-1 -o custom-columns=POD_IP:.status.podIPs
   ```
   NOTE: You should be able to SSH into any node in your cluster and use a tool
   such as curl to make queries against both IPs.

## Cleanup

Load Balancers cost money. Please be sure that they are cleaned up afterward.

1. Go to the `EC2 > Load Balancers`.
2. Select the same region as your EC2 instances are located and vew any load
   balancers associated with your Kubernetes Service.
3. Delete the Kubernetes Service: `kubectl delete -f svc-nginx-alb-1.yaml`
4. Wait a seconds to see if it is in fact deleted, if not,
   then manually delete it.
