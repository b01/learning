# Setup Tool Cluster Install

## You Should Know

* Should you make a mistake, you reset your cluster like so:

  ```shell
  # 1. Graceful kube-apiserver shutdown
  yq eval -i '.spec.containers[0].command = []' /etc/kubernetes/manifests/kube-apiserver.yaml
  timeout 60 sh -c 'while pgrep kube-apiserver >/dev/null; do sleep 1; done' || true
  # 2. Cleanup the cluster.
  sudo kubeadm reset -f --confg kubeadm-init-config.yaml
  # 3. Cleanup of network traffic rules.
  sudo docker run --privileged --rm registry.k8s.io/kube-proxy:v1.35.0 sh -c "kube-proxy --cleanup && echo DONE"
  # 4. Cleanup of CNI configuration.
  sudo rm -rf /etc/cni/net.d
  # 5. Cleanup of $HOME/.kube.
  rm -rf $HOME/.kube
  ```

  NOTE: Make sure to select the correct version of `kube-proxy` for your
  cluster.
