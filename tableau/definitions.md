# Definitions

* **Tableau site** - A grouping of your workbooks and other resources on a
server. A site has its own URL and users. It is walled off from other sites
on the server. A developer program sandbox is a site on the Tableau Cloud
server.
* **Tableau project** - A way of grouping workbooks and other resources within
a site. You can control access to those pieces by granting permissions at the
project level.
* **Extract** - a snapshot of data optimized for aggregation and loaded into
system memory to be quickly recalled for visualization.
