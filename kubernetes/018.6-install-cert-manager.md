# Install cert-manager

I'm not sure when to install this, I think before the aws-vpc-cni???

1. First, install `cmctl`:
   ```shell
   wget https://github.com/cert-manager/cmctl/releases/download/v2.4.1/cmctl_linux_amd64
   chmod +x cmctl_linux_amd64
   sudo mv cmctl_linux_amd64 /usr/local/bin/cmctl
   ```
2. Install `cert-manager`
   ```shell
   wget https://github.com/cert-manager/cert-manager/releases/download/v1.20.0/cert-manager.yaml

   kubectl apply -f cert-manager.yaml
   ```
3. Join the worker nodes;
   4. Copy the join template from the samples
      [kubeadm-dual-stack-join-node-aws.yaml] on each worker node
      ```shell
      getIps () {
          iface=${1}
          export IPV4_ADDR="$(ip -4 route get 8.8.8.8 | awk '{print $7}')"
          export IPV6_ADDR="$(ip -6 route get 2607:f8b0:4009:816::200e | awk '{print $11}')"
          echo "IPV4_ADDR=${IPV4_ADDR}"
          echo "IPV6_ADDR=${IPV6_ADDR}"
      }
      getIps "ens5"
      KUBE_TOKEN="<token>"
      KUBE_TOKEN_SHA="<token-sha>"
      envsubst < kubeadm-dual-stack-join-node-aws.yaml.tmpl | sudo tee kubeadm-dual-stack-join-node-aws.yaml
      sudo kubeadm join --config kubeadm-dual-stack-join-node-aws.yaml
      ```
